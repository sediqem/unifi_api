## 🔍 What is unifi_api?

unifi_api is a tool that helps you work with UniFi Dream Machine devices. It connects to your network hardware and security cameras. You can check on your devices, clients, networks, firewalls, cameras, and sensors. This tool uses simple commands to get or send information to your UniFi Dream Machine. It works with versions 10.1.84 for Network and 6.2.88 for Protect.

This app is built on Elixir, a programming language known for reliability and speed. You do not need to know Elixir to use unifi_api. The software handles the details in the background, so you can focus on your network and devices.

## 📋 Using unifi_api for the First Time

Once open, the app needs to connect with your UniFi Dream Machine.

1. **Provide your device details**  
   Enter the IP address of your UniFi Dream Machine or UniFi Protect system.  
   (You can find this in your router settings or UniFi app.)

2. **Enter your login info**  
   Use the username and password for your UniFi controller. This lets unifi_api access device data.

3. **Choose what to view**  
   The app will show options like sites, devices, clients, cameras, and sensors. Select what you want to explore.

4. **Explore device data**  
   You can see network status, client connections, camera feeds, and firewall rules. The app updates information in real time.

## ❓ Troubleshooting Common Issues

If you have problems, try these fixes:

- **App won’t open**: Restart your computer and try again.  
- **Cannot connect to UniFi device**: Check if the IP address and login info are correct. Make sure your PC is on the same network.  
- **Slow data updates**: Confirm your internet connection is stable.  
- **Installation fails**: Run the installer as administrator. Right-click the setup file and select “Run as administrator.”  
- **Firewall blocking app**: Allow unifi_api through your Windows firewall in the firewall settings.
